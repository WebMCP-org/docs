TanStack Start just works with [`@mcp-b/react-webmcp`](/packages/react-webmcp). The `useWebMCP` hook handles SSR safety internallyâ€”no special configuration needed.

```tsx "routes/index.tsx"
import { createFileRoute } from '@tanstack/react-router';
import { useWebMCP } from '@mcp-b/react-webmcp';
import { z } from 'zod';

export const Route = createFileRoute('/')({
  component: Home,
});

function Home() {
  useWebMCP({
    name: 'greet',
    description: 'Greet a user',
    inputSchema: { name: z.string() },
    handler: async ({ name }) => `Hello, ${name}!`,
  });

  return <div>Home</div>;
}
```

For tools that persist across navigation, register them in `__root.tsx`.

For complex SSR patterns (nested layouts, context providers, embedded agents), see the [Next.js guide](/frameworks/react) which covers React SSR in depth.
