## Welcome to WebMCP

**WebMCP is a polyfill for the W3C Web Model Context API** - a standards-track proposal that lets websites expose tools to AI agents through `navigator.modelContext`.

Add AI agent support to your website with a few lines of code. Your existing JavaScript functions become tools that AI assistants can discover and use, all while respecting user authentication and permissions.

## What is WebMCP?

WebMCP implements the `navigator.modelContext` web standard, letting you expose your website's functionality to AI agents. Your existing JavaScript functions become discoverable tools that AI assistants (like Claude, ChatGPT, or custom agents) can call directly.

### Why Use WebMCP?

- **Standards-Based**: Implements the W3C Web Model Context API proposal
- **Zero Backend**: Runs entirely in the browser - no server changes needed
- **Respects Auth**: Tools inherit your user's session and permissions
- **Framework Agnostic**: Works with React, Vue, vanilla JS, or any framework
- **Developer Friendly**: Simple API with React hooks for easy integration
- **Type Safe**: Full TypeScript support with Zod validation

## How It Works

1. **Load the polyfill**: Add `@mcp-b/global` to your website
2. **Register tools**: Use `registerTool()` or the `useWebMCP()` hook
3. **AI agents discover tools**: The MCP-B extension automatically detects your tools
4. **Users interact**: AI assistants can now call your tools to help users

## Quick Example

**React (Recommended)**

Use the `useWebMCP` hook for automatic tool registration with Zod validation:

```tsx
import '@mcp-b/global';
import { useWebMCP } from '@mcp-b/react-webmcp';
import { z } from 'zod';

function MyComponent() {
  useWebMCP({
    name: 'get_page_info',
    description: 'Get current page information',
    inputSchema: {
      includeUrl: z.boolean().optional()
    },
    handler: async ({ includeUrl }) => {
      return {
        title: document.title,
        ...(includeUrl && { url: window.location.href })
      };
    }
  });

  return <div>My Component</div>;
}
```

**Vanilla JavaScript**

Use `registerTool()` to register individual tools:

```javascript
import '@mcp-b/global';

navigator.modelContext.registerTool({
  name: "get_page_info",
  description: "Get current page information",
  inputSchema: {
    type: "object",
    properties: {
      includeUrl: { type: "boolean" }
    }
  },
  async execute({ includeUrl }) {
    return {
      content: [{
        type: "text",
        text: JSON.stringify({
          title: document.title,
          ...(includeUrl && { url: window.location.href })
        })
      }]
    };
  }
});
```
