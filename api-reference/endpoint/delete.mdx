---
title: 'Delete User'
api: 'DELETE /users/{id}'
description: 'This endpoint deletes a specific user'
---

### Path Parameters

<ParamField path="id" type="string" required>
  The ID of the user to delete
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Indicates if the deletion was successful
</ResponseField>

<ResponseField name="message" type="string">
  A confirmation message
</ResponseField>

<RequestExample>

```bash cURL
curl --location --request DELETE 'https://api.example.com/v1/users/usr_1234567890' \
--header 'Authorization: Bearer YOUR_API_KEY'
```

```javascript JavaScript
const response = await fetch('https://api.example.com/v1/users/usr_1234567890', {
  method: 'DELETE',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY'
  }
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.delete(
    'https://api.example.com/v1/users/usr_1234567890',
    headers={
        'Authorization': 'Bearer YOUR_API_KEY'
    }
)

print(response.json())
```

</RequestExample>

<ResponseExample>

```json Response
{
  "success": true,
  "message": "User usr_1234567890 has been successfully deleted"
}
```

</ResponseExample>
