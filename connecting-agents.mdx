## Local MCP Clients

Connect desktop AI apps (Claude Desktop, Claude Code, Cursor) to your browser tools:

### Native Server Bridge

The MCP-B native server bridges the extension to local MCP clients:

```bash
npm install -g @mcp-b/native-server
@mcp-b/native-server
```

Configure your MCP client:

```json
{
  "mcpServers": {
    "webmcp": {
      "type": "streamable-http",
      "url": "http://127.0.0.1:12306/mcp"
    }
  }
}
```
