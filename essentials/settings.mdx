## Configuration File

All global settings are configured in the `mint.json` file at the root of your documentation.

## Basic Settings

```json
{
  "name": "Your Documentation",
  "logo": {
    "dark": "/logo/dark.svg",
    "light": "/logo/light.svg"
  },
  "favicon": "/favicon.svg"
}
```

## Colors

Customize your brand colors:

```json
{
  "colors": {
    "primary": "#0D9373",
    "light": "#07C983",
    "dark": "#0D9373",
    "anchors": {
      "from": "#0D9373",
      "to": "#07C983"
    }
  }
}
```

## Navigation

Configure your documentation structure:

```json
{
  "navigation": [
    {
      "group": "Get Started",
      "pages": ["introduction", "quickstart"]
    },
    {
      "group": "API Reference",
      "pages": ["api-reference/authentication", "api-reference/endpoints"]
    }
  ]
}
```

## Top Bar

Add links and CTAs to your top navigation:

```json
{
  "topbarLinks": [
    {
      "name": "Support",
      "url": "mailto:support@example.com"
    }
  ],
  "topbarCtaButton": {
    "name": "Dashboard",
    "url": "https://dashboard.example.com"
  }
}
```

## Footer

Configure footer links and social media:

```json
{
  "footerSocials": {
    "x": "https://x.com/example",
    "github": "https://github.com/example",
    "linkedin": "https://www.linkedin.com/company/example"
  }
}
```

## Tabs

Create tabbed navigation:

```json
{
  "tabs": [
    {
      "name": "API Reference",
      "url": "api-reference"
    },
    {
      "name": "SDKs",
      "url": "sdks"
    }
  ]
}
```

## Analytics

Integrate analytics platforms:

```json
{
  "analytics": {
    "ga4": {
      "measurementId": "G-XXXXXXXXXX"
    },
    "mixpanel": {
      "projectToken": "your-project-token"
    }
  }
}
```

## SEO

Optimize for search engines:

```json
{
  "metadata": {
    "og:title": "Your Documentation",
    "og:description": "Comprehensive documentation for your product",
    "og:image": "https://example.com/og-image.png",
    "twitter:card": "summary_large_image"
  }
}
```

<Check>
  After updating `mint.json`, your changes will be reflected when you refresh your development server or deploy to production.
</Check>
