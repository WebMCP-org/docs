## Navigation Structure

The navigation is configured in `mint.json` using groups and pages:

```json
{
  "navigation": [
    {
      "group": "Get Started",
      "pages": ["introduction", "quickstart", "installation"]
    }
  ]
}
```

## Groups

Groups organize related pages together in the sidebar:

```json
{
  "group": "API Reference",
  "pages": [
    "api-reference/authentication",
    "api-reference/errors",
    "api-reference/rate-limits"
  ]
}
```

## Nested Navigation

Create nested groups for complex documentation:

```json
{
  "navigation": [
    {
      "group": "Getting Started",
      "pages": [
        "introduction",
        {
          "group": "Setup",
          "pages": ["setup/installation", "setup/configuration"]
        }
      ]
    }
  ]
}
```

## Tabs

Separate major sections with tabs:

```json
{
  "tabs": [
    {
      "name": "Documentation",
      "url": "docs"
    },
    {
      "name": "API Reference",
      "url": "api-reference"
    },
    {
      "name": "SDKs",
      "url": "sdks"
    }
  ]
}
```

## Anchors

Add quick links with icons:

```json
{
  "anchors": [
    {
      "name": "Documentation",
      "icon": "book-open-cover",
      "url": "https://docs.example.com"
    },
    {
      "name": "Community",
      "icon": "slack",
      "url": "https://slack.example.com"
    },
    {
      "name": "Blog",
      "icon": "newspaper",
      "url": "https://blog.example.com"
    }
  ]
}
```

## Page Metadata

Each page can have metadata in its frontmatter:

```yaml
---
title: 'Page Title'
description: 'Page description for SEO'
sidebarTitle: 'Short Title'
icon: 'rocket'
---
```

## Hidden Pages

Hide pages from navigation while keeping them accessible:

```yaml
---
title: 'Hidden Page'
mode: 'hide'
---
```

Or exclude from navigation in `mint.json`:

```json
{
  "navigation": [
    {
      "group": "Guides",
      "pages": ["guide1", "guide2"]
    }
  ],
  "hiddenPages": ["secret-page"]
}
```

## Icons

Add icons to navigation items:

```yaml
---
title: 'Settings'
icon: 'gear'
---
```

Available icon sets:
- Font Awesome
- Hero Icons
- Custom SVG icons
